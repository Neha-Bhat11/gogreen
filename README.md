🌱 GoGreen – Eco-Friendly Community Platform
 Project Overview

**GoGreen** is a web-based platform designed to promote environmental awareness and sustainable living. It connects users, encourages eco-friendly habits, and provides a space to share ideas, initiatives, and resources related to protecting the environment.
This project was developed as part of an academic final-year project.
 
 Objectives
* Encourage users to adopt eco-friendly practices
* Provide a platform for sharing environmental ideas and initiatives
* Promote community engagement for sustainability
* Manage users, categories, and activities efficiently

 Technologies Used
* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP

 Project Structure

```
gogreen/
│── admin/              # Admin panel files
│── user/               # User interface files
│── database/           # Database file
│    └── gogreen.sql
│── assets/             # CSS, JS, images
│── index.php           # Main entry point
│── config.php          # Database connection
```

 Installation & Setup
 Step 1: Clone the Repository

```
git clone https://github.com/your-username/gogreen.git
```
Step 2: Move to XAMPP Directory
* Copy the project folder to:
C:\xampp\htdocs\
 Step 3: Start Server

* Open XAMPP Control Panel
* Start:

  * Apache
  * MySQL

 Step 4: Import Database

1. Open phpMyAdmin
2. Create a new database named:

```
gogreen
```

3. Click **Import**
4. Select file:

```
database/gogreen.sql
```

5. Click **Go**

 Step 5: Run the Project

Open browser and go to:

```
http://localhost/gogreen
```
